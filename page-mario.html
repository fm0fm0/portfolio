馬力歐陪你喝一杯 Drink with Mario

Podcast 品牌網站 UI Redesign + 全站重構

  • 專案角色：UI優化+切版
  • 專案年份：2023
  • 裝置平台：Web(RWD)

專案背景

《馬力歐陪你喝一杯》是一個人物訪談 Podcast，以酒吧聊天或朋友聚會的氛圍，採訪各行各業的創意、創新專業人士分享有趣故事，主要收錄訪談內容並轉化為文章。本次任務為全站介面優化、提升使用體驗、切版重構。

資訊架構

設計規劃

依照品牌端、設計端、開發端 3 大面向，列出痛點及優化方案：

  • POINT
    01

    沒有 Style Guide，顏色混亂不一致

    遵循品牌精神，建立基礎的 Style Guide，包含顏色、字級、間距...等等

  • POINT
    02

    文字顏色對比度不足，不易閱讀

    提升文字對比度，調整背景色，提升閱讀體驗

  • POINT
    03

    卡片元件佈局圖文容易跑版

    重新設計元件提升擴充性與彈性

視覺規劃

設計系統

設定主題色彩並自定義 Bootstrap 的 theme color 搭配使用。

首頁 frist view 優化

  • hero區塊與最新節目區塊都是輪播且區塊重疊，佈局不太合理，新版調整成分開的佈局，更加簡潔。
  • 卡片佈局彈性不足容易破版，新版省略時間及按鈕，調整標題文案可有有更大容許度的排版。

節目卡片優化

  • 評估卡片訊息以標題和圖片最吸引讀者點閱，捨棄只呈現集數和節目長度的版型，調整成可完整呈現標題和圖片的形式。
  • 在首頁時此區塊的需求是連結至列表頁而非在此換頁跳轉，所以移除頁碼改成顯示看更多按鈕。

聽眾回饋輪播優化

  • 因文字量差距過大造成換頁時上下跳動劇烈，撐開容器高度並保持內容置中來改善。
  • 手動輪播不容易發現，調整為自動輪播並設定合理的輪播秒數方便閱覽。

footer 優化

整併footer及收聽平台區塊使畫面更精簡，並使用彩色的收聽平台icon來呈現更具吸引力。

節目列表封面優化

  • 內容維護者在上傳圖片後從後台編輯人像容易被裁切，數度調整最佳顯示高度呈現最佳化顯示。
  • 節目標題及描述文字壓在圖上閱讀辨識度不佳，調整為放在純色背景更利於閱讀。

節目列表頁碼優化

頁碼的狀態不明顯，以及意味不明的標示文字，調整成簡單易懂且明確的分頁頁碼。

聽眾報告頁內容優化

舊版將一整張圖片置入文章，不利於SEO搜尋，新版將區塊開分重新安排佈局和繪製圖表，採用和網站主題的色系整體更佳和諧。
