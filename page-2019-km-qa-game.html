農學大哉問 KM Q&A game

知識問答遊戲網站 Web Design

  • 專案角色：網頁設計師
  • 專案年份：2019
  • 裝置平台：Web(RWD)

專案背景

透過簡單的問答遊戲和誘人的獎品，引導用戶進入農業知識的精彩世界。我們的目標是幫助民眾透過參與遊戲來「學習新知」，同時探索我們網站上的主題館資訊內容。我的工作涵蓋了視覺設計、版面設計、切版和社群媒體宣傳素材的製作。

專案目標

  1. 農業知識入口網的希望透過舉辦活動刺激新會員成長。
  2. 透過問答遊戲讓玩家了解吸收知識，並帶動農業知識入口網的觀看人次。
  3. 活動對象為一般民眾，預期網站活動為期1個月。

資訊架構

視覺規劃

  1. 如何讓遊戲設計的簡單易懂，讓玩家能夠快速上手
  2. 如何讓畫面豐富有趣吸引玩家提高黏著度？

解決方案

  1. 每題遊戲題組選項都是一顆超大的按鈕，字級稍大，在手機版也能輕鬆點擊，不論是低年齡層或高年齡層皆友好。
  2. 參考益智節目的視覺設計風格，繽紛活潑的配色提高畫面的吸睛度。

設計系統

設定主題色彩並自定義 Bootstrap 的 theme color 搭配使用。

標準色 / 標準字 / ICON / 按鈕

使用多彩的色調增添活力及豐富的氛圍，標準文字則使用網頁高支援率的黑體。

主視覺 / logo / 插圖

使用插畫使氣氛變得繽紛有趣，並提高畫面的吸睛度，並壓縮圖片最佳化以提升網頁效能
