專案背景

「關鍵評論網」品牌至今已經發展出中文版、English Edition和日本版三種語系版本。為了保持各語系版本的一致形象，我們已經在中文版首頁進行了視覺翻新。為了進一步提升國際版的品質進行改版計畫，以中文版為參照，同步設計語彙，同時保留現有的網站架構並最小化對網站程式的修改，以實現優化動線和視覺翻新的目標。慶祝成立10週年，logo也同步更新。

設計規劃

調整的方向上依影響範圍分為全域範圍與局部頁面範圍：

全域範圍規劃

  • POINT
    01

    Style Guide 不夠完整，只有品牌色深中淺三色不足以應用到全站

    建立 style guide遵循品牌色定義 Primary Color 及灰階，擴充 Color Scale 提升設計系統泛用性

  • POINT
    02

    有襯線體字體於數位裝置閱讀體驗不佳，全站標題及內文皆為襯線體

    改為無襯線字型選用適合品牌風格的無襯線字體作為主要字體，並選用襯線字體作為標題裝飾使用

  • POINT
    03

    文字顏色對比度不佳，閱讀困難，部分字級設定無規範

    提高對比度提高深色及淺色背景時的文字的對比度，建立字及規範，提升閱讀體驗

  • POINT
    04

    中文版版型套用英文語系後，字距與行距需要微調成舒適閱讀的留白空間

    建立 style guide中文與英文字元因筆劃複雜度不同，所需要的閱讀留白空間是不同的

局部頁面規劃

  • POINT
    01

    會員專區的頁籤看不出來是可以被點擊的，資料沒有設計空值狀態

    空值狀態設計增加空資料狀態時的設計，調整頁籤排版，讓空間有效被運用

  • POINT
    02

    選單佈局可擴充性低，改版後選單選項變多，原先的佈局無法容納

    擴充性高的選單設計調整成與中文版相同的樣式，增加未來擴充性。

  • POINT
    03

    feature區塊結構複雜，裝飾用的效果不明顯且難以模組化開發

    feature卡片設計簡化在整體設計不做大幅變動的前提下，調整成複用性高的元件模組，可應用於多個頁面上

  • POINT
    04

    文章列表頁的文章卡片數量規則不一致且數量過多，視覺負擔較重

    改為統一的卡片數量將卡片數量統一，增加維護便利性，全站的列表頁眾多數量一致管理

資訊架構

視覺原則

  • 直覺的動線引導

    資訊處理分級明確，色彩簡化達到視覺降噪，讓使用者可以更直觀的操作

  • 元件狀態明確

    設計顯而易見的操作狀態，例如：按鈕的default/hover/active..等

  • 品牌識別形象導入

    依據「關鍵評論網」新版 LOGO 導入品牌藍色及視覺風格，強化品牌印象

色彩計畫

顏色的定義，以LOGO的品牌色為基礎去擴充色階，選用的灰色是偏藍色調的灰階讓整體更一致。顏色越單純越能讓網站豐富的內容被凸顯出來。

字體計畫

字體方面，我認為在擁有巨量文章的媒體新聞網站是非常重要的，會大大影響閱讀體驗。需要傳達專業性、信賴感、穩定度，紙張媒體中使用襯線字體可以讓印刷品在閱讀上文字更清晰，但在數位媒體中使用電子媒體閱讀體驗更佳，基礎字體的定調為無襯線字體，保留傳統媒體的使用有襯線體傳達專業嚴肅的調性，在區塊性標題上選用有襯線體，選用各作業系統支援性較佳的內建字體英文字體提升載入速度。

首頁翻新

選單優化

選單因為資訊量的擴增，需要合理的排版讓使用動線順暢，層次明確

會員專區優化

會員專區先前的頁籤設計不明顯，僅顯示純文字的樣式，難以直覺辨識那是一系列可以切換的頁籤按鈕。空資料狀態時畫面沒有被撐開，空間也沒有有效運用，考量日後有擴增的可能性，將選單改成左邊，主要內容在右邊的格局

文章頁優化

原先的排版左側區塊大量留白RWD，為了提升閱讀舒適體驗，特別為文章頁的文字行距及字距做細部校調，變得寬鬆一些，文章寬度縮限在較窄的範圍可以更容易閱讀。

文章列表模組化

列表頁是所有文章的集合，改版前的痛點有hashtag 過多收納後不被點擊的問題，和每個列表頁的卡片數量皆不一致，統整成一致的數量，對讀者而言分頁顯示後每個頁面的卡片變短，贊助標記位置，

一般來說文章列表的頁碼常見為十位數，關鍵評論網的文章數量非常大，頁碼多達千位數，會需要合理的設計安排。視覺上讓當前頁面的頁碼狀態更明確，操作上觸控面積增加讓使用者操作更容易點擊

作者列表頁優化

作者列表頁原先有自定義背景設計，在需求確認時發現，作者們幾乎沒有使用這個功能，推測作者們擅長文字撰寫但沒有製圖能力，經討論後移除功能，但樣式上保留預設背景圖樣並刷淡縮小面積僅為裝飾用，原先作者介紹的區塊較分散零散，也一併作整併和集中，目標讓讀者可以更快閱覽到下方的文章列表，閱讀更多文章增加觸擊率。

狀態頁設計

網頁出現找不到頁面或伺服器錯誤等回應會讓使用者感到挫折、困惑、不耐煩等情緒出現，除了讓訊息明確傳達之外，添加與品牌風格相符富含人文感的幽默，減緩使用者的負面感受，適切導引使用者之外也增加趣味性。

原先的設計是用海倫凱勒 Helen Keller 的名言：

"When one door of happiness closes, another opens; but often we look so long at the closed door that we do not see the one which has been opened for us."

通往幸福之門的概念，狀態碼 404 not found 代表找不到頁面，而我們幫讀者打開新的大門，覺得富饒趣味便繼續沿用此概念，並在右下角加上了一個小門，滑鼠移入後門會打開，讓使用者有一個小發現而感到有趣。

原本這頁是沒有設計，僅顯示系統預設，為了讓整體有一致性，背景換成米開朗基羅的《創世紀》，狀態碼500 Internal Server Error 代表伺服器錯誤回應。《創世紀》中男子與先知寓意為本站與讀者之間的的連結，這只是個暫時性的狀態，正在努力修復中精彩內容很快就會與讀者見面。
