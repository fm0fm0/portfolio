Carolsiya 平價質感飾品 Carolsiya Accessoris

飾品電商

00. Sumnary 概述

這是一個Side project，個人創作非商用的創作網站，意在練習建構從無到有的過程，透過練習來增加經驗值與鍛鍊思考路徑，從網站規劃、UI設計、切版、上線全部一條龍包辦。

前往網站 →

01. Preliminary plan 初步規劃

  • 需擬定一個主題，這個主題得要好研究資料又好找素材。
  • 想嘗試設計不那麼中規中矩的layout看自己是否真的能切版成功。

02. Pain Point 痛點分析

  • 該如何定下網站類型及主題風格？
  • 一般專案皆有團隊做前置作業後續在按照規劃進行作業，該如何自己從無到有開始一項專案呢？

03. Solution 解決方案

  • 從自身愛好著手發想，決定想做自己熟悉有興趣且圖片素材好搜集的主題：飾品。想做出帶有浪漫文藝氣質、富有詩意的氛圍感。
  • 想給自己一個稍具難度的類型，於是選定購物型網站，內容涵蓋產品介紹、會員註冊登入、購物車流程，實用度也高，因為沒有團隊做前置規劃，於是將曾經有過規劃購物型網站的經驗整合運用其中，並大量閱覽各大購物型網站、電商平台網站，研究其中的架構組成、購物流程與圖文細節，做出基本網站架構。

04. Design Systerm 設計系統

設定主題色彩並自定義 Bootstrap 的 theme color 搭配使用。

標準色 / 標準字 / 按鈕

大面積使用白色與粉色為基調，並使用半透效果，輔助色為深灰色，創造出一種朦朧氛圍，標準文字使用明體增添文藝感。

標題 / 插圖

使用草寫作為標題，增加浪漫詩意氛圍，半透粉色系疊加於圖片上，更添一抹朦朧，八角形的裝飾線條俐落，帶有一點簡約又古典的味道。

05. The Outcom 結論

當初光是想要做什麼主題就傷透了腦筋。從無到有的過程，自行規劃網站架構及定義文案內容、自選圖文素材及編修 選定創作購物型網站是因為對於靜態網站來說，除了經常會在購物網站亦是熟悉的題材，是參考資料多、比較好研究且較有挑戰性，其中需要思考的細節頗多，包含購物車流程、會員登入/註冊，將使用者購物時的情境感帶入，思考版位的安排，研究各大購物型網站的版面呈現，在RWD時如何佈局，購物車流程怎麼樣規劃最順暢，這完整的過程經驗過後讓我更加熟悉且有成就感，是我自己很喜歡的作品。
